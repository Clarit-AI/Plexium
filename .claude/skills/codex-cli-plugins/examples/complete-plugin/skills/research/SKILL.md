# Research Workflow

Conduct structured research on a given topic.

## Process

1. **Clarify scope**: Confirm the research question and any constraints (date range, domain, depth).
2. **Search**: Use web search to find relevant sources. Prioritize recent, authoritative results.
3. **Synthesize**: Summarize key findings with source citations.
4. **Organize**: Present results in a structured format with sections for key findings, open questions, and recommended next steps.
5. **Cite**: Include source URLs as markdown hyperlinks.

## Output Format

```
## [Topic] — Research Summary

### Key Findings
- Finding 1 (Source)
- Finding 2 (Source)

### Open Questions
- Question 1
- Question 2

### Recommended Next Steps
- Action 1
- Action 2

### Sources
- [Title](URL)
```
