# Plexium Claude Plugin

This plugin provides Claude Code commands for:

- bootstrapping the `plexium` binary
- running `plexium setup claude`
- running `plexium verify claude`
- querying the wiki with `plexium retrieve`
- showing or applying the native Claude MCP command

Local install:

```text
/plugin marketplace add /path/to/Plexium
/plugin install plexium-tools@plexium-local
```
