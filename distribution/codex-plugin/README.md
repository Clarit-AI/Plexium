# Plexium Codex Plugin

This plugin keeps Codex on the Plexium happy path by teaching it to:

- bootstrap the `plexium` binary
- run `plexium setup codex`
- run `plexium verify codex`
- query the wiki with `plexium retrieve`
- show or apply the native Codex MCP command

Repo marketplace install uses:

- `<repo-root>/.agents/plugins/marketplace.json`
- `distribution/codex-plugin/`
